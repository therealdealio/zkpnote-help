# ZKPnote Help

User guides and documentation for ZKPnote — a privacy-first note-taking app with blockchain-backed **Proof of Originality** and a peer-to-peer marketplace.

## What's New in v0.5.0

- **Wallet Account Tracking** — Every Phantom and seed-phrase wallet is auto-registered with a deterministic username so the admin dashboard can see the full active user base
- **Link Account for Phantom** — Phantom users can now reserve a custom username and optional email address without providing a seed phrase
- **Share Link Sign-Up** — Readers opening a share link without a wallet can create a full account (username + email + password) in one step and view the shared note immediately
- **MCP Batch Save** — New `save_notes` tool writes multiple notes in a single vault pull/push — much faster for bulk imports from Claude
