# ZKPnote Help

User guides and documentation for ZKPnote — a privacy-first note-taking app with blockchain-backed **Proof of Originality** and a peer-to-peer marketplace.

## What's New in v0.3.0

- **Proof of Originality** — Prove you were the first to create a note by registering its SHA-256 hash on Solana. One click from the editor toolbar.
- **Verify Page** — Public page at `/verify` where anyone can check if content has been proved on-chain. No login required.
- **Persistent Phantom Sessions** — Phantom wallet sessions survive page refreshes. No more re-approving the signature message on every reload.
